# Урок 4: Структуры, Методы и Указатели

## Введение
В Go нет классов в привычном понимании (как в JS `class`). Вместо них используются **структуры (structs)** для данных и **методы** для поведения.

## 1. Структуры (Structs)
Как объекты в JS, но со строгой схемой.
```go
type User struct {
    ID    int
    Name  string
    Email string
}
```

## 2. Указатели (Pointers) — Важно!
В отличие от JS, где объекты передаются "по ссылке", в Go всё передается **по значению** (копируется). Чтобы изменить оригинал, нужен указатель.
- `&v` — взять адрес переменной.
- `*p` — разыменовать (получить значение по адресу).

## 3. Методы
Функции, привязанные к типу.
- **Value Receiver**: `func (u User) Greet() { ... }` (работает с копией).
- **Pointer Receiver**: `func (u *User) UpdateEmail(e string) { ... }` (меняет оригинал).

## Сравнение с JS
| Концепция | JavaScript | Go |
|-----------|------------|----|
| Объект/Класс | `class User { ... }` | `type User struct { ... }` |
| Поля | `this.name = name` | `u.Name = name` |
| Методы | `getName() { ... }` | `func (u User) GetName() { ... }` |
| Ссылки | Автоматически для объектов | Явно через указатели `*type` |
