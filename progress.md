# Прогресс обучения Go

## Пройденные уроки (полностью, с практикой):
- **Урок 1**: Установка Go, Hello World, команды `go mod init`, `go run`, `go build`. Бинарник без расширения — норма. ✅
- **Урок 2**: 
  - Переменные: `var`, `:=`, `const`, `iota`, zero values.
  - Типы: int/float/string/byte/rune/bool.
  - Контроль потока: if (с init), for (единственный цикл), switch.
  - Enum-паттерны: slice/map[type].String().
  - Практика: `variables.go`, `sumEven(100)=2550`. ✅

## Текущий урок (в процессе):
- **Урок 3**: Функции — базовые, variadic (`...`), множественные возвраты (`value, err`), closures (factory funcs), `defer` (LIFO), подробные отличия от JS (явная сигнатура, no defaults, multi-returns, defer LIFO).
  - Дополнения: variadic примеры, аналогии JS rest/try-catch.
  - Задание: avg, safeDivide, generator, 3 defer'а. **Ожидаю код+вывод от тебя**.

- **Пройдено**: ~30%.
